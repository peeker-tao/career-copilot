# Career-Copilot 用户故事文档

> 版本：v1.1 | 日期：2026-06-27

## 一、概述

本文档以用户故事（User Story）的格式，描述 Career-Copilot 平台各功能模块的业务需求。每个用户故事遵循标准模板：

> **作为** `<角色>`，**我希望** `<功能>`，**以便** `<业务价值>`。

### 状态说明

| 标记 | 含义 |
|:----:|------|
| ✅ 已实现 ⏳ 待测试 | v1.0 MVP 功能，已完成开发，尚未完成测试验收 |
| 🆕 v1.1 新增 ⏳ 待测试 | v1.1 新增功能，已完成开发，尚未完成测试验收 |

---

## 二、用户角色定义

| 角色 | 编码 | 描述 |
|------|------|------|
| 求职者 | **USR** | 在校大学生、应届毕业生、社招求职者，使用平台进行面试模拟和职业规划 |
| 访客 | **VIS** | 未登录用户，可浏览平台介绍但无法使用核心功能 |
| 管理员 | **ADM** | 系统维护者，管理用户、日志和平台数据审核 |

## 五、非功能需求

| 编号 | 类型 | 描述 |
|------|------|------|
| NFR-001 | **性能** | AI 面试题生成响应时间不超过 5 秒 |
| NFR-002 | **性能** | 页面首次加载时间不超过 3 秒 |
| NFR-003 | **可用性** | 系统支持 7×24 小时运行，可用性 ≥ 99.5% |
| NFR-004 | **安全** | 用户密码加密存储（bcrypt），JWT Token 鉴权 |
| NFR-005 | **兼容性** | 支持 Chrome、Edge、Safari 最新版本 |
| NFR-006 | **可扩展性** | 后端模块化设计，支持后续增加新岗位类型和面试题型 |
| NFR-007 | **数据隐私** | 用户简历数据加密存储，未经授权不分享给第三方 |
