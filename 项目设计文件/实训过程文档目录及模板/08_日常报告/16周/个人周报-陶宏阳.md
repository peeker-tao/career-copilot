## 四、技术亮点与成果

### 亮点 1：LLM 多 Provider 适配器

```
LLMProvider (interface)
  ├── OpenAICompatibleProvider (gpt-4o-mini)
  ├── 通义千问 Provider (qwen-turbo)
  └── DeepSeek Provider (deepseek-chat)
```

工厂模式自动检测环境变量，切换模型无需修改业务代码。

### 亮点 2：面试流式对话引擎

- WebSocket 实时通信，`ai_message_chunk` 事件实现打字机效果
- AI 判断逻辑：评分(1-5) → 追问/换题 → 结束，全自动
- 对话历史持久化到 `interview_messages` 表

### 亮点 3：简历多级解析

```
PDF上传 → pdf-parse 文本提取 → LLM 结构化解析
        ↕ (回退)
    扫描件 → tesseract.js OCR → LLM 结构化解析
```

### 亮点 4：Prompt 工程

5 个专业化 Prompt 模板，严格控制 JSON 输出格式，确保 AI 行为可预测。

## 七、心得体会
