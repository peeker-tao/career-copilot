# career-copilot

具体切入点：解决大学生和求职者缺乏面试经验、不了解岗位要求，难以进行有效职业规划

的问题。

创新点：AI模拟面试官数字人，能根据目标岗位（如“产品经理”、“算法工程师”），生成高度

仿真的面试问题，并能根据回答进行追问和反馈。同时，模型能分析用户简历和技能，结合

市场数据，推荐个性化的职业发展路径和学习计划。
